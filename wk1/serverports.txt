1. What is the definition of a server port?
A server port is a logical communication endpoint used by a server to receive and send network traffic for a specific service or application.

2. What is server port range?
The server port range is from 0 to 65535, which represents all possible TCP and UDP ports.

3. How many bits does number above represent?
The port number range represents 16 bits.
